# Spiral Matrix

Write a function that accepts an integer N and returns a NxN spiral matrix.

## Examples
```python
matrix(2)
# output
[[1, 2],
[4, 3]]
```

```python
matrix(3)
# output
[[1, 2, 3],
 [8, 9, 4],
 [7, 6, 5]]
```

```python
matrix(4)
# output
[[1,   2,  3, 4],
 [12, 13, 14, 5],
 [11, 16, 15, 6],
 [10,  9,  8, 7]]
```
